normalize.css

发表于 2021-09-25 | 更新于 2022-02-25

软件简介

https://github.com/necolas/normalize.css/
Normalize.css 是一个可以定制的CSS文件，它让不同的浏览器在渲染网页元素的时候形式更统一。

应用

Normalize.css 能干什么?

保留有用的默认值，不同于许多 CSS 的重置
标准化的样式，适用范围广的元素。
纠正错误和常见的浏览器的不一致性。
一些细微的改进，提高了易用性。
使用详细的注释来解释代码。
支持的浏览器：
Google Chrome (latest)
Mozilla Firefox (latest)
Mozilla Firefox ESR
Opera (latest)
Apple Safari 6+
